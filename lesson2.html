🌍 HTML5 Semantic Elements

💡 Note: Semantic elements in HTML5 describe the **meaning and structure** of content, making pages easier for both developers and browsers to understand. Examples include <header>, <nav>, <main>, <article>, <section>, and <footer>.

🔗 Navigation Bar

The <nav> tag defines navigation links to other pages or sections. The list items are displayed inline using CSS.

🧠 About Article Section:

✅ Summary:
• <header> → Introduces content or navigation.
• <nav> → Contains site navigation links.
• <main> → Represents the main content of the page.
• <article> → Self-contained content (like a blog post).
• <section> → Divides content into logical parts.
• <figure> + <figcaption> → Used for images with captions.
• <footer> → Footer section of the page.
• <time> → Defines a date or time in machine-readable format.
